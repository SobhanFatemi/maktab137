Flexbox and CSS Grid Layout

It's okay to combine CSS Grid Layout and Flexbox.

Is the content in a straight line or column (cards, navigation items, or other content blocks)? For occasions that require elements to line up in one direction, remaining in more “one-dimensional”, typically Flexbox is the better option. Also, Flexbox is good at dynamically scaling elements. Maybe you've tried this in a row of boxes, by adding display:flex on a parent element and flex properties on the child elements, there's a nice line formed and this is an efficient way to make sure they are all the same height.

**If things do not look right, make sure your browser is in “Experimental Mode”.
